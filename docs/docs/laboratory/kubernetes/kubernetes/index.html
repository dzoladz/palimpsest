3-Node Cluster Computing Hardware

K3s

The lightweight, certified Kubernetes distribution K3s was selected. The K3s distro is designed from ARM64 architectures like the Raspberry Pi 4.

Docker Compose to Kubernetes

Kompose is a command-line conversion tool. It takes a Docker Compose file and translates it into Kubernetes resources. There’s also a GUI version maintained on GitHub.

Basic Conversion to Single File

kompose convert -f docker-compose.yml -o kubernetes-output.yml

Convert to Helm Chart

kompose convert -c
